About

I'm a PhD student and researcher at INTIA & i3Lab (University of Extremadura), working where AI meets software engineering.

My research focuses on quality assurance for code produced by large language models. I work with Specification-Driven Development (SDD) so that formal specifications are not just prompts for generation, but a basis for tests, checks, and evaluation artifacts that make LLM-generated code easier to validate and trust.

The broader goal is rigorous, transparent AI-assisted development: models can draft code quickly, but we still need systematic ways to measure and improve its correctness.

**TL;DR:** [Awesome GitHub Copilot](https://awesome-copilot.github.io/) es un repositorio público que centraliza agentes, instrucciones, skills y herramientas comunitarias para exprimir todo el potencial de GitHub Copilot. Ya hay 182 agentes, 246 skills, y contando.

> Nota: este post fue redactado por un agente mientras Emi investigaba el repositorio, revisaba los números y mantenía el entusiasmo bajo control.

## Un catálogo ordenado para Copilot

GitHub Copilot creció tan rápido que necesitaba una tienda. Así nace **Awesome Copilot**: un sitio que recopila y organiza extensiones, agentes especializados, instrucciones de best practices y workflows automatizados que la comunidad ha estado publicando.

![Awesome Copilot homepage](/blog/2026-03-24-awesome-copilot/Captura%20de%20pantalla%202026-03-24%20a%20las%2012.23.16.png)

Como ves en la captura, el ecosistema es amplio:

- **182 Agentes**: copilots personalizados para tareas específicas
- **173 Instrucciones**: guías de coding standards y best practices
- **246 Skills**: paquetes autocontenidos con instrucciones y recursos
- **60 Plugins**: complementos organizados por tema
- **7 Workflows**: automatizaciones con GitHub Actions
- **6 Hooks**: eventos automatizados mediante agentes
- **10 Tools**: servidores MCP y herramientas de desarrollo
- **Learning Hub**: artículos y guías para dominar Copilot

## Por qué importa una lista así

Antes: buscas en Google, abres 15 pestañas, discutes con tu navegador si sobrevivirá, y a mitad del camino olvidas qué querías.

Ahora: entras a Awesome Copilot, buscas por categoría, ves ratings comunitarios, y en 5 minutos sabes si algo te sirve.

Es organización. Es fricción reducida. Es diferencia entre "¿por dónde empiezo?" y "voy directo a probar esto".

## Qué encontrás ahí

Si quieres:

- **mejorar prompts** → instrucciones y guías documentadas
- **automatizar workflows** → agentes y skills listos para usar
- **especializar Copilot** para tu dominio → agentes comunitarios ya testeados
- **integrar con tu stack** → plugins y tools
- **aprender en profundidad** → Learning Hub con tutoriales

Lo más importante: cada recurso viene con descripción, repo, y usualmente ejemplos. No es tirar código al aire esperando que pegue.

## El juego cambió

Copilot ya no es una herramienta solitaria. Es un ecosistema. Y como todo ecosistema, creció y necesitó cierta organización.

**Awesome Copilot** es esa organización. Un catálogo comunitario que dice: "Aquí están los mejores recursos, pruébalos, contribuye, comparte lo que descubriste".

Si usás Copilot regularmente y querés ir más allá de autocomplete, es un lugar al que volver constantemente.
